How It Works

1

User Discovers Tracker

Your local proxy detects a tracking cookie or script on a website you visit.

2

Report to Network

The discovery is anonymously reported to the PrivacySpace central server.

3

Instant Broadcast

Server validates and broadcasts the tracker to ALL connected users via WebSocket.

4

Everyone Protected

All users' blocklists update automatically. The tracker is now blocked for everyone!

5

Public Dashboard

The discovery appears on the live dashboard, exposing tracking companies publicly.
